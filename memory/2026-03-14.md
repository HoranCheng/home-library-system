### Wi-Fi Monitoring (single USB-C port solution)
- iPad Pro 2020 has only 1 USB-C port — can't monitor + charge simultaneously via USB
- Solution: pair over USB once → enable "Wi-Fi Sync" on iPad → USB free for charger
- `battery.js` updated: tries `ideviceinfo -n` (Wi-Fi) first, falls back to USB
- New endpoints: `GET /api/battery/wifi-check`, `POST /api/battery/pair`
- Full chain: iPad ←(Wi-Fi)→ Mac reads battery | iPad ←(USB-C)→ charger ←(power)→ Tapo

### 毒蘑菇 Ray Marching Shader (commit `b51bef0`)
- New file: `public/stress/raymarch.html`
- Technique: SDF (Signed Distance Field) ray marching — same as 毒蘑菇 benchmark
- Scene: multiple SDF mushrooms (stem + cap + gills + spots), ground, particles
- Lighting: 6 dynamic lights + soft shadow marching + AO
- 4 quality levels: 轻度(64 steps, 0.5x res) → 极限(160 steps, 1.5x res)

- Added to stress menu and dashboard scenario list
