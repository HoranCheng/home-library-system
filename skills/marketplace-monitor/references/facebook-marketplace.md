# Facebook Marketplace Notes

## Preconditions
- User is logged in.
- User opens the target Marketplace search page.
- If browser relay is required, user attaches the tab first.

## Practical search variants
Try in order:
1. `mac mini m4`
2. `apple mac mini m4`
3. `mac mini m4 16gb`
4. `mac mini m4 512`

If sparse:
- widen radius (50km -> 80km)
- include nearby hubs in keyword (e.g., `Melbourne`, `Geelong`)
- temporarily broaden to `mac mini` and re-filter manually for M4

## False-positive filters
Exclude unless user asks:
- keyboards/mice/cases/cables
- old Intel Mac mini generations
- wanted posts (WTB)
- obvious scams (price too low + no detail + no location)

## Quick sanity checks
- Price vs typical range looks plausible
- Listing has real photos and usable description
- Location present
- URL opens to item detail

## Suggested buyer follow-up template
- Is this still available?
- Exact spec please (chip, RAM, SSD)?
- Any repairs or issues?
- Can you share serial-prefix / purchase proof?
- Pickup suburb and best time?
