## 统一录入流程

### T-021: 录入流程重构 — 扫码+手动合一
当前：点 + → 进入录入页，有 ISBN 输入 + 扫码按钮 + 手动表单
目标：
- [ ] 统一入口：点 + 直接打开相机（扫码为主路径）
- [ ] 底部保留手动 ISBN 输入框（扫码失败时快速切换）
- [ ] 扫码成功 → 自动查询 → 显示预览卡片（封面+书名+作者）
- [ ] 预览卡片上可编辑分类、位置
- [ ] 一键保存 → 回到扫码继续录入
- [ ] 长按或点击手动按钮 → 展开完整手动录入表单

### T-022: 录入预览卡片 UI
- [ ] 新的预览卡片设计：封面左 + 信息右（类似漫游卡片风格）
- [ ] 元数据来源标签（Google Books / Open Library / 手动）
- [ ] 分类选择：芯片式快速选择 + 自定义输入
- [ ] 位置选择：记忆上次位置 + 快速切换
- [ ] 保存动画：卡片缩小飞入书架

### T-023: 连续扫码优化
- [ ] 保存后相机不关闭，直接继续扫
- [ ] 录入计数器（本轮已录 N 本）
- [ ] 振动反馈区分：扫码成功 vs 保存成功
- [ ] 重复 ISBN 检测提前到扫码阶段
