二、当前卡住的问题
================================

1. PWA 部署仍然 404
   - sw.js 和 manifest.json 在线上一直 404
   - 根因：GitHub Pages 可能设置成 "Deploy from branch" 模式
     而我们的 Actions workflow 用的是 artifact 模式
   - 最简解决：你在 GitHub 网页 Settings → Pages 确认一下
     Source 是 "GitHub Actions" 还是 "Deploy from a branch"
   - 如果是 branch 模式，那只需要跑 npm run deploy 就行
     （gh-pages -d preview 会正确同步所有文件）

2. Google OAuth 客户端 ID 未设
   - 前端 Google 登录按钮已隐藏
   - 等你准备好了跑：wrangler secret put GOOGLE_CLIENT_ID
